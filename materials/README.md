# Materials

Syllabus, practice, and assessment files for **Computer Information**.

Teaching content is provided in **three formats** where applicable:

| Format | Use |
|---|---|
| `.md` | Git-friendly source |
| `.txt` | Easy reading / quick review |
| `.docx` | Word / printing / sharing |

## Layout

| Path | Description |
|---|---|
| [`syllabus/`](./syllabus/) | Official syllabus DOCX + extracted md/txt/docx |
| [`qa/`](./qa/) | Tiered question banks (md + txt + docx) |
| [`assignments/`](./assignments/) | Assignments & projects (md + txt + docx) |
| [`labs/`](./labs/) | Practical / lab exercises (md + txt + docx) |
| `glossary.md` / `.txt` / `.docx` | Course glossary |
| `mindmap.md` / `.txt` / `.docx` | Concept mind map |
| `revision-sheet.md` / `.txt` / `.docx` | Rapid revision sheet |
