## Must-Remember Lists by Unit

### T1 — Basic Computing (Obj 1, 2, 9)

- **Computer:** electronic device; data → information under instructions.
- **IPO:** Input, Process, Output, Storage.
- **OS:** manages hardware; runs applications; file/security services.
- **Hardware / software / peopleware** — one example each for a law office.
- **Internet:** browser → URL → server; bookmarks for court/DB portals.
- **Digital communication:** professional email; VC; confidentiality in subject/body.
- **AI survey:** history (symbolic → statistical → deep learning); legal apps (research, analytics, drafting); **not** only chatbots.

### T2 — MS Word (Obj 3, 4, 9)

- **Styles** over manual formatting; Heading 1/2, Normal.
- **Headers/footers:** advocate details, case title, page numbers.
- **Tables:** fee schedules, timelines.
- **Mail merge:** Excel list + Word template + merge fields + preview.
- **Citations:** footnotes or bibliography—consistent style.
- **Track Changes + comments:** senior review; accept/reject.
- **Notice:** heading, facts, legal basis, demand, signature.
- **Affidavit:** numbered facts, verification, attestation.
- **Agreement:** parties, recitals, terms, termination, signatures.

### T3 — MS Excel (Obj 5, 9)

- **Workbook** = file; **worksheet** = tab.
- Formulas start with `=`; never type totals manually.
- **SUM, AVERAGE, IF, COUNTIF, SUMIF** — know one use each.
- **VLOOKUP/XLOOKUP:** link Client ID → name/rate.
- **Sort** reorders; **Filter** hides non-matching rows.
- **Charts:** case status, billing—label axes.
- **Case tracker:** ID, court, status, hearing date, advocate.
- **Billing:** hours × rate; paid/unpaid; outstanding total.
- **No merged cells** in data tables.

### T4 — MS PowerPoint (Obj 6, 9)

- **One idea per slide**; large readable font; high contrast.
- **Moot structure:** title → facts → issues → arguments → authorities → prayer.
- **Theme/template** for consistency.
- **SmartArt:** processes (FIR steps), court hierarchy.
- **Charts:** only with source note.
- **Animations/transitions:** few and purposeful.
- **Speaker notes** for advocacy slides.

### T5 — AI Ethics (Obj 2, 7, 8, 9)

- **Legal research AI:** faster search; **verify** in primary sources.
- **Generative AI:** creates text; risk of **hallucination** (fake citations).
- **Judiciary/analytics:** prediction/triage—transparency and bias concerns.
- **AI drafting:** outline only with human rewrite and sign-off.
- **Ethics triangle:** privacy, bias, accountability.
- **Human-in-the-loop:** mandatory for client/court work.
- **Limitations:** no professional judgment; jurisdiction gaps; confidentiality rules.
- **Never** paste confidential client files into public AI.

## Common Pitfalls (Exam & Practical)

| Unit | Pitfall | Correct approach |
|------|---------|------------------|
| T1 | Confusing application software with OS | OS = Windows/Linux; Word/browser = applications |
| T1 | Treating AI as infallible research | Verify every citation in official DB |
| T2 | Spaces/tabs instead of styles | Apply styles; use multilevel list |
| T2 | Forgetting page numbers for filings | Footer page number from insert |
| T2 | Mail merge without data source | Link Excel/CSV before insert fields |
| T3 | Typing totals instead of formulas | `=SUM()`, `=E2*F2`, etc. |
| T3 | Merged cells in case register | One row per case; wrap text |
| T4 | Wall of text on one slide | Split; bullets; speaker notes |
| T4 | Excessive animation | ≤50% slides; content over effects |
| T5 | Client data in public ChatGPT | Redact/fictional; use approved tools |
| T5 | Filing AI text uncited/unverified | Human review + source check |

## Rapid-Fire 15 (with Answer Keys)

1. **What are the four IPO stages?**  
   **A:** Input, Process, Output, Storage.

2. **One function of an operating system?**  
   **A:** Manages hardware/resources and runs application programs (file management acceptable).

3. **Track Changes is used for?**  
   **A:** Recording and reviewing document revisions in collaborative legal drafting.

4. **Workbook vs worksheet?**  
   **A:** Workbook = entire Excel file; worksheet = single sheet/tab inside it.

5. **Formula for sum of C1:C20?**  
   **A:** `=SUM(C1:C20)`

6. **Mail merge needs what two components?**  
   **A:** Main document template + data source (e.g., Excel list).

7. **Name two sections of an affidavit.**  
   **A:** Any two: numbered statement of facts, verification, attestation/signature.

8. **FILTER vs SORT in Excel?**  
   **A:** Sort reorders all rows; filter displays subset matching criteria.

9. **What is an AI hallucination?**  
   **A:** Confident but incorrect AI output (e.g., fake case citation).

10. **One limitation of AI in law?**  
    **A:** Any one: hallucination, lack of accountability, jurisdictional error, cannot replace advocate judgment.

11. **Purpose of headers/footers in Word?**  
    **A:** Repeated identification and pagination on each page for professional filings.

12. **SmartArt is best for?**  
    **A:** Visualizing processes, hierarchies, or relationships (e.g., court structure, FIR steps).

13. **Human-in-the-loop means?**  
    **A:** Qualified human reviews and takes responsibility for AI-assisted work.

14. **COUNTIF used for?**  
    **A:** Counting cells that meet a single condition (e.g., unpaid invoices).

15. **Three ethical risks of pasting client file into public AI?**  
    **A:** Confidentiality breach, data-protection violation, unreliable/hallucinated legal content used without verification.
