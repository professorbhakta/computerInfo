# Part A — Word as a Legal Workstation

## 1. Why MS Word Matters for BA-LLB Students

A first internship week often includes:

- Formatting a **legal notice** on letterhead style  
- Preparing an **affidavit** with correct margins and verification  
- Editing a senior’s draft using **Track Changes**  
- Making a **table of dates / annexures**  
- Sending personalised letters through **Mail Merge**  
- Converting final draft to **PDF** for filing / email

Students who only “type text” struggle. Students who control **styles, layout, review tools, and structure** look practice-ready.

### Law Office Lens

Senior: “Send me a clean draft.”  
Meaning: correct formatting, consistent fonts, numbered paragraphs, no messy spacing, named file, and preferably Track Changes if you edited an existing draft.

## 2. Getting Started — Creating and Saving Documents

### 2.1 Create a New Document

1. Open Microsoft Word.  
2. Choose **Blank document** (or a template if office provides one).  
3. Immediately **Save As** with a professional name.

### 2.2 Professional File Naming (Revision from Unit 1)

Good: `2026-07-13_LegalNotice_Patel_to_ABCPvtLtd_Draft-v1.docx`  
Bad: `notice final final.docx`

### 2.3 Save Formats Lawyers Commonly Use

| Format | Use |
|---|---|
| `.docx` | Editable working draft |
| `.pdf` | Filing / sharing final version |
| `.doc` | Older compatibility (legacy) |
| Template `.dotx` | Reusable office formats |

### Key Definitions

- **Document:** A Word file containing text, tables, images, and formatting.  
- **Template:** A reusable starter layout (styles, margins, letterhead placeholders).  
- **Soft copy / Hard copy:** Digital file / printed paper.

## 3. Creating and Formatting Documents

### 3.1 Fonts and Readability for Legal Work

Common professional choices in many Indian offices:

- Body: **Times New Roman** or **Georgia** / similar serif (court preference varies)  
- Some corporate teams use **Calibri / Arial**  
- Size often **12 pt** for body; headings larger or bold  
- Line spacing often **1.5** for drafts (follow local court/office rules)

**Rule:** Consistency beats decoration. Do not mix five fonts in one petition.

### 3.2 Essential Formatting Tools

| Tool | What it does | Legal use |
|---|---|---|
| Bold / Italics | Emphasis | Party names, case titles, Latin phrases sparingly |
| Alignment | Left / Centre / Justify / Right | Cause title centred; body often justified |
| Bullets / Numbering | Lists and numbered paragraphs | Petition paragraphs, prayer clauses |
| Increase/Decrease Indent | Hierarchy | Sub-paragraphs (a)(b)(c) |
| Show/Hide ¶ | Reveals hidden formatting marks | Fix mysterious spacing issues |
| Format Painter | Copy formatting | Match style across sections quickly |
| Styles (Normal, Heading) | Structured formatting | Long agreements / academic papers |

### 3.3 Paragraph Discipline

Legal documents fail visually when:

- Extra blank lines are random  
- Tabs are used instead of proper indents  
- Space before/after paragraph is inconsistent  
- Manual numbering is typed (`1.` `2.`) and breaks during edits

Prefer **automatic numbering** for petition paragraphs.

### 3.4 Find and Replace (High-Value Skill)

Use **Ctrl + H** to:

- Replace wrong party spellings everywhere  
- Change “Petitioner” to “Applicant” carefully in the right context  
- Clean double spaces

### Exam Trap

Formatting is not only “looking nice.” In exams and viva, explain **purpose**: readability, professionalism, court expectations, easier review.

### Law Office Lens — Cause Title Block

A clean cause title usually needs:

- Court name  
- Jurisdiction / case type placeholders  
- Party names with “Versus”  
- Document title (e.g., LEGAL NOTICE / AFFIDAVIT)

Centre alignment + controlled line spacing matters here.

## 4. Page Layout for Legal Documents

### 4.1 Key Page Layout Settings

| Setting | Meaning | Typical legal caution |
|---|---|---|
| Margins | Space from page edge | Courts/offices may prescribe minimum margins |
| Orientation | Portrait / Landscape | Most pleadings portrait; schedules sometimes landscape |
| Size | A4 / Letter | India commonly **A4** |
| Columns | Multi-column text | Rare in pleadings; more for brochures |
| Breaks | Page / Section breaks | New exhibit sets, annexure separation |
| Widow/Orphan control | Avoid single stranded lines | Cleaner printed petitions |

### 4.2 Section Breaks vs Page Breaks

- **Page Break:** Force next content to new page (same section settings).  
- **Section Break:** Allows different headers/footers/margins/orientation in parts of one file.

### Draft Lab

Set a practice notice:

- A4  
- Margins: reasonable even margins (follow your college/court instruction sheet if any)  
- Portrait  
- Page numbers in footer

### Key Definitions

- **Margin:** Blank space surrounding the printable area.  
- **Section:** A portion of a document that can have independent layout settings.  
- **Page layout:** Arrangement of page size, margins, orientation, and related structure.

### Exam Trap

Students confuse **Enter Enter Enter** with correct spacing. Prefer paragraph spacing settings and page breaks over dozens of blank lines.

### Quick Check

1. Difference between page break and section break.  
2. Why A4 + consistent margins matter for filing sets.

## 5. Headers and Footers

### 5.1 Meaning

- **Header:** Content in top margin area (letterhead text, matter name, “CONFIDENTIAL”).  
- **Footer:** Content in bottom margin area (page number, firm name, file reference).

### 5.2 Common Legal Uses

| Placement | Examples |
|---|---|
| Header | Office name, “WITHOUT PREJUDICE”, matter short title |
| Footer | Page X of Y, advocate contact, draft version label |
| Different first page | Letterhead on page 1 only |
| Different odd/even | Book-style printing (less common in pleadings) |

### 5.3 Page Numbers

Insert → Page Number options:

- Bottom centre / bottom right (office preference)  
- Format page numbers (i, ii, iii for preliminary pages; 1, 2, 3 for main body) using sections when needed

### Law Office Lens

A 40-page agreement without page numbers is a review nightmare. Page numbers are professionalism, not decoration.

### Draft Lab

Create footer: page number + your name.

# Part B — Structure Tools: Tables, Mail Merge, References

## 6. Tables in Legal Documentation

### 6.1 Why Lawyers Use Tables

- Chronology of dates  
- Schedule of property / payments  
- Party details  
- Annexure index  
- Comparison of clauses  
- Fee / expense summaries (often later also in Excel)

### 6.2 Core Table Skills

1. Insert table (rows × columns)  
2. Merge / split cells (for titles spanning columns)  
3. Borders and shading (clean, not colourful)  
4. Text alignment inside cells  
5. AutoFit (window / contents)  
6. Header row repeat for multi-page tables (useful in long schedules)

### 6.3 Formatting Taste for Law

- Prefer simple grid borders  
- Avoid heavy colours in pleadings  
- Keep font consistent with body text  
- Leave readable cell padding

### Sample: Annexure Index Table

| Annexure | Description | Pages |
|---|---|---|
| A | Copy of agreement dated … | 1–8 |
| B | Legal notice dated … | 9–11 |
| C | Postal receipt | 12 |

### Key Definitions

- **Table:** Grid of rows and columns to organise information.  
- **Cell:** One box in a table.  
- **Merge cells:** Combine adjacent cells into one.

### Exam Trap

A table is not only for “decoration.” Explain that it improves clarity of schedules and chronologies in legal drafting.

## 7. Mail Merge for Legal Practice

### 7.1 Definition

> **Mail Merge** combines a main document with a data source (like a contact list) to generate personalised letters, notices envelopes, or labels in bulk.

### 7.2 Mail Merge Components

1. **Main document** — the notice/letter template with placeholders  
2. **Data source** — Excel/CSV/Outlook list with names, addresses, amounts  
3. **Merged result** — individualised outputs

### 7.3 Typical Law Office Uses

- Bulk legal notices to multiple allottees / borrowers / members  
- Hearing date intimation letters  
- Fee reminders  
- Client update letters with personalised names

### 7.4 Basic Steps (Conceptually)

1. Prepare main document text.  
2. Link data source.  
3. Insert merge fields (`<<Name>>`, `<<Address>>`, `<<Amount>>`).  
4. Preview results.  
5. Finish & merge (print / edit individual docs).

### Law Office Lens — Ethics Warning

Bulk merge is powerful. Verify:

- Correct recipient list  
- Correct cause facts per person (do not merge wrong dues/amounts)  
- Confidential data handling  
- That automated tone still meets professional standards

### Key Definitions

- **Merge field:** Placeholder replaced by data from each record.  
- **Data source:** File/list supplying variable information.  
- **Record:** One row of recipient data.

### Quick Check

1. Define Mail Merge.  
2. Give two legal use-cases and one risk.

## 8. References and Citations

### 8.1 Why This Matters

BA-LLB students write:

- Assignments and seminar papers  
- Research notes  
- Memorials (to an extent)  
- Intern research memos

Word tools help manage footnotes, endnotes, captions, and bibliographies.

### 8.2 Useful Reference Tools

| Tool | Purpose |
|---|---|
| Footnotes | Notes at page bottom (common in academic writing) |
| Endnotes | Notes at document end |
| Captions | Label tables/figures |
| Cross-reference | Link to headings, figures, footnotes |
| Table of Contents | Auto-generate from styles |
| Bibliography / Citations manager | Academic citation support (style-dependent) |

### 8.3 Legal Citation Caution

Court pleadings and notices often follow **local drafting conventions**, not always academic citation managers. For class assignments, follow your faculty’s required citation style.

### 8.4 Styles + TOC Connection

If you apply **Heading 1 / Heading 2** styles to agreement article titles, Word can build an automatic Table of Contents — extremely useful in long contracts.

### Draft Lab

Write a one-page mini essay paragraph and insert one footnote explaining a term (e.g., “Cause of action”).

### Exam Trap

Do not claim every legal document must use APA/MLA. Distinguish **academic citation** vs **legal drafting format**.

# Part C — Collaboration & Legal Drafting Workflows

## 9. Track Changes and Comments

### 9.1 Definitions

> **Track Changes** records insertions, deletions, and formatting edits so reviewers can accept or reject them.  
> **Comments** are side notes for questions, instructions, or suggestions without altering the main text directly.

### 9.2 Why Law Offices Depend on This

- Senior reviews junior draft  
- Co-counsel collaborates  
- Client suggests factual corrections  
- Version accountability (“who changed clause 5?”)

### 9.3 Practical Workflow

1. Turn on **Review → Track Changes**.  
2. Edit normally; markup appears.  
3. Reviewer uses **Accept / Reject**.  
4. Resolve **Comments**.  
5. Save final clean copy (and keep a marked copy if needed).

### 9.4 Display Modes

- Simple Markup / All Markup / No Markup / Original  
Understand these for viva/practical exams.

### Law Office Lens

Never silently rewrite a senior’s draft without Track Changes unless instructed. Invisible edits destroy trust.

### Key Definitions

- **Markup:** Visual record of edits.  
- **Accept/Reject:** Finalise or discard a proposed change.  
- **Balloon comments:** Comment display in margin area.

### Quick Check

1. Difference between Track Changes and Comments.  
2. Why is “No Markup” view risky while editing?

## 10. Legal Drafting Using MS Word — Professional Method

### 10.1 Drafting Mindset

Word skills support legal mind, not replace it. Process:

1. Gather facts and instructions  
2. Identify document type and audience  
3. Structure headings/parts  
4. Draft in clear numbered paragraphs  
5. Format for readability  
6. Review with checklist  
7. Export PDF / print sets as required

### 10.2 Universal Structure Habits

- Title clearly stated  
- Parties identified early  
- Facts in chronological order where relevant  
- Legal grounds / obligations separated from facts  
- Clear prayer / demand / operative clauses  
- Signature / verification blocks with space

### 10.3 Style Consistency Checklist

- Same font and size throughout body  
- Consistent date format (`13 July 2026` or `13.07.2026` — pick one)  
- Consistent party naming (`the Client`, `the Vendor`)  
- Numbered paragraphs stable after edits  
- No decorative clipart in pleadings

### 10.4 From Draft to Filing-Ready

1. Spellcheck + human proofing of names/dates/amounts  
2. Confirm annexure labels match references in text  
3. Page numbers present  
4. Export PDF  
5. File naming for final version (`..._Final.pdf`)

# Part D — Document Types: Notices, Agreements, Petitions, Affidavits

## 11. Preparation of Legal Notice in MS Word

### 11.1 What is a Legal Notice? (Student-level)

A formal written communication sending information of grievances/demands and often warning of legal action if not addressed — commonly used before certain civil actions (context-specific).

### 11.2 Typical Word Layout Blocks

1. Sender advocate/office details  
2. Date  
3. Recipient details  
4. Subject line  
5. Salutation  
6. Facts in numbered paragraphs  
7. Legal basis / breach alleged (carefully)  
8. Demand / relief sought with timeline  
9. Consequences of non-compliance (professionally worded)  
10. Closing & signature  
11. Copy to (if any)

### 11.3 Formatting Tips

- Formal tone; avoid slang  
- Numbered facts  
- Bold subject line  
- Adequate signature space  
- Keep a PDF + dispatch proof later (postal/email records)

### Sample Skeleton (Illustrative only — not a substitute for senior-approved drafting)

```text
[Letterhead / Advocate details]

Date: ...

To,
[Recipient Name & Address]

Subject: Legal Notice regarding ...

Sir/Madam,

Under instructions from my client ..., I state as under:

1. ...
2. ...
3. ...

You are hereby called upon to ... within ... days ...

Yours faithfully,
[Advocate Name]
```

### Exam Trap

In Computer Information exams, they may ask **steps/features in Word** used to prepare a notice, not only the legal theory of notices.

### 13.1 Student-Level Understanding

### 13.2 Common Word Tasks While Preparing Petitions

### 13.3 Practical Formatting Tips

- Avoid huge walls of unnumbered text

## 14. Preparation of Affidavits in MS Word

### 14.1 Meaning (Classroom Level)

An affidavit is a written statement of facts sworn/affirmed before an authorised officer. It supports pleadings and applications.

### 14.2 Typical Parts to Format Cleanly

1. Cause title / reference  
2. Deponent introduction  
3. Numbered factual statements  
4. Statement as to truth  
5. Verification  
6. Place & date  
7. Signature / attestation space  
8. Notary/oath officer formatting as required locally

### 14.3 Word Tips

- Numbered paragraphs for each fact  
- Clear spacing for signatures and seals  
- Avoid editing substance after notarisation conceptually (office process discipline)  
- Keep a clean PDF of notarised scan in matter folder (Unit 1 habit)

### Exam Focus

Be ready to list **MS Word features** used: fonts, numbering, page layout, header/footer, page breaks, tables (if schedule included), export to PDF.

## 19. Full Model Answers (Exam Style)

### Model Answer — Features of MS Word useful for legal documentation (8–10 marks)

Microsoft Word is widely used in legal offices for drafting and formatting documents such as notices, agreements, petitions, and affidavits. Important features include:

1. **Text formatting tools** for professional fonts, alignment, and emphasis.  
2. **Paragraph numbering and styles** for structured pleadings and clauses.  
3. **Page layout settings** (A4, margins, orientation) for filing standards.  
4. **Headers and footers** for letterhead information and page numbers.  
5. **Tables** for chronologies, schedules, and annexure indexes.  
6. **Mail Merge** for bulk personalised communications.  
7. **Track Changes and Comments** for collaborative review between juniors and seniors.  
8. **References tools** such as footnotes and table of contents for academic/long documents.  
9. **Export to PDF** for stable final versions.

Thus, MS Word supports both the substance and presentation of legal writing when used with professional discipline.

### Model Answer — Track Changes (5–7 marks)

Track Changes is a Review feature that records insertions, deletions, and formatting modifications. Reviewers can accept or reject each change. Comments allow marginal queries without directly altering text. In law offices, this ensures transparency, preserves drafting history, and prevents silent unauthorised edits. After finalisation, a clean version can be saved for execution or filing, while a marked version may be retained for negotiation history.

### Model Answer — Mail Merge in legal practice (5–7 marks)

Mail Merge combines a template document with a data source to produce personalised outputs at scale. Advocates may use it for bulk notices or intimation letters. The process involves preparing the main document, connecting a recipient list, inserting merge fields, previewing, and completing the merge. Accuracy of data and confidentiality are critical because errors can send wrong demands to wrong persons.

## 24. Deep Dive — Styles Are a Lawyer’s Secret Weapon

### Why juniors ignore Styles
They seem “academic.” In reality, Styles save hours in long agreements.

### Practical style map for an agreement
| Style | Use |
|---|---|
| Title | Agreement name |
| Heading 1 | Article I, II, III |

## 25. Headers/Footers — Advanced but Useful Scenarios

1. **Draft watermark-like caution in header:** `INTERNAL DRAFT — NOT FOR FILING`  
2. **Matter code in footer:** `Matter: 2026/CIV/0142`  
3. **Section-based numbering:** Annexures start at page 1 again, or continue sequence — decide intentionally.  
4. **Confidentiality label** on every page of sensitive opinions.

### Common error
Typing page numbers manually (`Page 1`, `Page 2`) as ordinary text. When content shifts, numbering collapses. Always use Insert Page Number fields.

## 27. Mail Merge — Mini Case Study

A housing society counsel must send demand notices to 120 members with different dues.

**Correct approach**
1. Excel columns: Name, FlatNo, Address, AmountDue, DueDate.  
2. Word main document with polite formal language.  
3. Merge fields for Name/FlatNo/AmountDue/DueDate.  
4. Preview at least 5 random records.  
5. Senior spot-checks extremes (highest dues, zero dues, missing address).  
6. Merge to PDF/print sets.  
7. Store data source securely.

**Failure scenario**
Wrong column mapped → every notice shows the same amount. This is both a Word-skill error and a professional risk.

## 30. Full Model Answer — Preparing Petition in MS Word (10 marks)

A petition should be prepared with both legal structure and Word discipline. First, create an A4 document with appropriate margins and professional font. Format the cause title clearly, usually centred, identifying court, parties, and document nature. Draft facts and grounds in automatically numbered paragraphs for clarity and easy reference. Use page breaks before annexure sections and insert an annexure index table. Add page numbers through footer. If multiple people edit, enable Track Changes. After review, produce a clean version and export PDF for filing sets while retaining the editable `.docx` master. Consistency in names, dates, and annexure labels is essential because formatting errors can reduce readability and professionalism.

## 33. Extended Short Notes Bank (Ready Memorisation)

### Short Note: Header and Footer
Header is the top margin area; footer is the bottom margin area. They repeat information such as office name, confidentiality labels, and page numbers. “Different First Page” allows letterhead-style first page. In legal practice they improve navigation and professionalism of multi-page drafts.

### Short Note: Section Break
A section break divides a document into sections that can have distinct layout settings (headers, footers, margins, orientation). It is more powerful than a simple page break and is useful when annexures or schedules need different numbering/layout behaviour.

### Short Note: Legal Notice in MS Word
Preparing a legal notice involves formal layout blocks (sender, recipient, subject, numbered facts, demand, closing) and Word tools for formatting, numbering, headers/footers, and PDF export. Accuracy of facts and professional tone are as important as formatting.
